# Description
A command line employee manager app. Used to keep track of employees, roles and departments. Ability to add, view and remove sections from each category. 

# Installation:

__Install__

```
MySQL
MySQL Workbench
``` 

__Run inside of Workbench__

```
schema.sql 
seed.sql
``` 

__Run in Command Line__

```
npm install 
node server.js
``` 

__Follow prompts__


# Technologies: 

* JavaScript
* MySQL
* Node.js
